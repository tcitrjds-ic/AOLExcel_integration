# CSV選考データマージツール

3種類の選考CSV（**2次選考 / 最終選考 / 内定出し**）から「日時」と「氏名」を抽出し、
1つのCSVに集約するTkinter製のデスクトップツールです。

## セットアップ

```powershell
pip install -r requirements.txt
```

## 起動

```powershell
python csv_merger.py
```

## 使い方

1. アプリ起動後、画面に3つの枠（`＜2次選考＞` / `＜最終選考＞` / `＜内定出し＞`）が表示されます。
2. 各枠に対応するCSVファイルをドラッグ&ドロップします（枠あたり複数CSV可、1枠のみでもOK）。
3. 「マージしてCSV出力」ボタンを押すと、`~/Downloads/merged_YYYYMMDD_HHMMSS.csv` に保存され、
   エクスプローラーで該当ファイルが選択状態で開きます。

### 出力フォーマット

| 列 | 内容 |
| --- | --- |
| 日時 | `yyyy/M/d HH:mm` 形式に整形（パース不能な値は元の文字列のまま） |
| 姓 | 元の氏名を最初の空白で分割した前半 |
| 名 | 元の氏名を最初の空白で分割した後半（区切りが無い場合は空） |
| 種別 | `2次選考` / `最終選考` / `内定出し` |

## 抽出ルール

| 枠 | 日時 列名（優先） | 日時 位置（フォールバック） | 氏名 列名（優先） | 氏名 位置（フォールバック） |
| --- | --- | --- | --- | --- |
| 2次選考 | `2次選考` | C列 | `グループ選考 【総合評価】（ユーザ）` | H列 |
| 最終選考 | `最終選考会` | D列 | `最終 【総合評価】（ユーザ）` | J列 |
| 内定出し | `内定だし（＝面談）` | E列 | `内定通知 1.入社させたいか（ユーザ）` | L列 |

- **列名優先**: ヘッダーに該当列名が見つかればそれを使用
- **位置フォールバック**: 列名が一致しない場合、アルファベット位置で取得
- **姓名分割**: 全角スペース（U+3000）または半角スペースで分割。区切りが無い場合は全て姓列に格納
- **氏名が空白の行**は出力から除外
- **時刻自体は絶対に変更しません**（表記の整形のみ）

## 文字コード

- 入力: 自動検出（`utf-8-sig` → `cp932` → `utf-8` → `latin-1` の順で試行）
- 出力: `utf-8-sig`（BOM付き、Excelで文字化けせず開けます）
